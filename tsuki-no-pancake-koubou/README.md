# つきのパンケーキこうぼう 🥞🌕

4歳児向けの「光と影のお料理あそび」Webゲーム。
iPhone / iPad の縦画面・横画面のどちらでも遊べます。

丸いパンケーキを惑星に見立て、上からランプの光が当たります。
月みたいな丸い板やクッキー、フルーツが光を遮ると影ができて、
影と光でトッピングの様子が変わります。日食・月食の構造を、
手ざわりのあるお料理あそびとして体験できます。

## あそびかた

1. このフォルダで簡易サーバを起動してブラウザで開く
   (ESモジュールを使っているため `file://` 直開きでは動きません)

```bash
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

2. ▶ ボタンでスタート
3. 下の道具バーから好きな道具を選んでパンケーキにタッチ

### おこること

| そざい | ひなた | ひかげ |
| --- | --- | --- |
| 🧈 バター | じゅわっと溶けて金色の池になる | 溶けずにそのまま |
| 🍯 はちみつ | サラサラになり、あたたかい方へ流れる | ねばって止まる |
| 🍫 チョコ | ゆっくり溶けなおす | カチッと固まってキラッと光る |
| 🧂 こなざとう | 溶けて消える | 影の形だけ白く残る |

- 🍓 いちご・🫐 ベリー・🍌 バナナはパンケーキと一緒にくるくる回り、小さな影を落とします
- 🌕 おつきさまボードと 🍪 クッキーをドラッグしてパンケーキにかざすと、日食みたいな影ができます
- クッキーの影がパンケーキの真ん中にくると「にっしょく!」— 空が暗くなり星が輝きます
- ✋「くるくる」でお皿を手で回したり、はじいて回したりできます
- 🐰 月うさぎの店長をタップするとくすぐれます
- うさぎからの「おねがい」ミッションを達成すると ⭐ がもらえます (失敗はありません)

## フォルダ構成

```
tsuki-no-pancake-koubou/
├── index.html          エントリポイント
├── css/style.css       UIスタイル (縦横対応・子ども向け大ボタン)
├── js/
│   ├── main.js         起動・レイアウト・メインループ
│   ├── state.js        共有状態・定数・座標変換・イベントバス
│   ├── scene.js        星空・地球・月面・テーブル・ランプ
│   ├── pancake.js      回転皿とパンケーキの描画
│   ├── light.js        光量計算・影・日食・コロナ
│   ├── occluders.js    おつきさまボードとクッキー
│   ├── butter.js       バターの溶けシミュレーション
│   ├── honey.js        はちみつの流れシミュレーション
│   ├── choc.js         チョコの固まりシミュレーション
│   ├── sugar.js        粉砂糖グリッド (影のスタンプ)
│   ├── fruits.js       いちご・ベリー・バナナ
│   ├── particles.js    きらきら・じゅわじゅわ・紙ふぶき
│   ├── rabbit.js       月うさぎの店長「もちた」
│   ├── missions.js     おねがいミッション
│   ├── audio.js        WebAudio合成の効果音とオルゴールBGM
│   ├── input.js        タッチ入力 (マルチタッチ対応)
│   └── ui.js           道具バー・ボタン・ミッションカード
└── test/smoke.mjs      Playwrightスモークテスト
```

## 技術メモ

- 依存ライブラリなしの素の Canvas 2D + WebAudio (音源ファイル不要)
- 光量は遮蔽物ごとのソフトシャドウを解析的に合成 (`lightAt`)
- 粉砂糖は 112×112 のローカル座標グリッドで管理し、行を分割して段階更新
- パンケーキの焼き目・お皿はオフスクリーンに事前生成して毎フレーム回転描画
- iOS対策: `touch-action: none`、`viewport-fit=cover`、初回タップで AudioContext 起動
