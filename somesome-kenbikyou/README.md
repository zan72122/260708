# そめそめけんびきょう 🔬💧

4歳児向けの「HE染色ごっこ」Webゲームです。案内役の **しずくちゃん** といっしょに、
プレパラート（標本）を液にひたして染め、顕微鏡でできあがりをのぞきます。
**成功／失敗はありません。** つけかたのちがいで、仕上がりの色あいがほんのり変わるのを楽しみます。

## あそびかた

1. `index.html` をブラウザで開く（スマホ・タブレットのタッチ操作OK）。
2. ▶ ボタンで はじめる。
3. HE染色の8工程を、しずくちゃんの合図にあわせて進める。
   - **つける工程**：プレパラートを びん の中へドラッグし、そのまま **長おし** すると
     どんどん染まります（リングがたまる）。指をはなすと止まります。← つける時間が仕上がりに反映。
   - **みずあらい**：3つのおみずから えらぶ（水質）。
   - **ふきふき＆フタ**：スポンジのところを何回もタップして水をふきとり、フタをのせる（脱水の十分さ）。
4. 顕微鏡ビューでできあがりを鑑賞し、📥 で **ずかん** にコレクションできる。

## HE染色の8工程

脱パラ → 水洗 → ヘマトキシリン → 分別 → ブルーイング → エオジン → 脱水 → 封入（ふきふき＆フタ）

## 「微妙な仕上がり」に効く7つの要素

| 要素 | ゲーム内の操作 | 仕上がりへの影響 |
|------|----------------|------------------|
| ヘマトキシリン時間 | むらさきの液に つける長さ | 核のむらさきの濃さ |
| 分別時間 | 分別の液に つける長さ | 核の色を少しもどす（うすく） |
| ブルーイング | あおくする液に つける長さ | 核が青寄りになる |
| エオジン時間 | ピンクの液に つける長さ | 細胞質のピンクの濃さ |
| 脱水時間 | 脱水の液に つける長さ | くっきり具合 |
| 脱水の十分さ | ふきふきのタップ回数 | モヤ（かすみ）の少なさ |
| 水質 | みずあらいで えらぶ | 発色のすなおさ |

顕微鏡ビューは毎回すこしだけ模様がちがいます（同じ標本はずかんに種として保存され、同じ絵で再現されます）。

## 技術・構成

- 追加のビルド不要。**HTML + CSS + 素の JavaScript** の単一ページゲーム。
- 画像・音声の外部ファイルは使わず、絵は **SVG / Canvas**、音は **Web Audio API** で生成。
- 保存は `localStorage`。

```
somesome-kenbikyou/
├── index.html          # 画面のがわ
├── css/
│   └── style.css       # パステル手描き風スタイル
├── js/
│   ├── audio.js        # BGM＋効果音（Web Audio）
│   ├── art.js          # しずくちゃん等のSVG
│   ├── staining.js     # 7要素 → 仕上がり色の計算
│   ├── microscope.js   # 顕微鏡ビュー描画（Canvas）
│   ├── gallery.js      # 標本コレクション（localStorage）
│   ├── steps.js        # HE染色8工程のデータ
│   └── game.js         # ゲーム進行のまとめ役
├── test/
│   └── smoke.mjs       # Playwrightの動作確認テスト
└── README.md
```

## 動作確認（開発者向け）

```bash
cd somesome-kenbikyou
npm install
npx playwright install chromium
npm test        # 8工程を自動走破し、結果・ずかん保存・コンソールエラーを確認
```
